
Get Forcast Notes

1. Go to the site below to get latitude & longitude. See GeoCoding.pdf in this directory for more information.
	
	https://geocoding.geo.census.gov/geocoder/locations/onelineaddress?address=4600+Silver+Hill+Rd%2C+Suitland%2C+MD+20746&benchmark=9&format=json

	https://geocoding.geo.census.gov/geocoder/locations/address?street=5+caladonia+ct&zip=61704&benchmark=9&format=json

2. Enter the latitude and latitude to get the grids 
	
	ttps://api.weather.gov/points/{latitude},{longitude}
	
	For example: https://api.weather.gov/points/39.7456,-97.0892

	https://api.weather.gov/points/40.45777,-88.93429

3. Get the forcast using the grids
	
	https://api.weather.gov/gridpoints/{office}/{grid X},{grid Y}/forecast
	
	For example: https://api.weather.gov/gridpoints/TOP/31,80/forecast

	https://api.weather.gov/gridpoints/TOP/70,86/forecast
